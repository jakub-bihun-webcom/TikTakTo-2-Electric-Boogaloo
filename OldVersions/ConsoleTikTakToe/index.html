Folgende Schritte sind notwendig um das Spiel zu spielen:

Öffne die Konsole mit:

Mac: command + option + J

Windows: Control + Shift + J

Folgende Commands werden benötigt:

- choosePlayerStart([Spieler]) = um den Spieler auszuwählen, der starten soll

-> move([Feld]]) = Setzt eine Markierung für den Spieler, der grade am Zug ist

- resetBoard() = um das Spiel auf die Anfangsposition zu setzen

- stateOfBoard() = um dir den aktuellen Spielstatus anzeigen zu lassen
