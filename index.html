Tic Tac Toe

Folgende Schritte sind notwendig um das Spiel zu spielen:

Öffne die Konsole mit
Mac
command + option + J
Windows
Control + Shift + J
Folgende Commands werden in der Konsole benötigt
move([Feld]])
Setzt eine Markierung für den Spieler, der grade am Zug ist
resetBoard()
Setzt das Spiel auf die Anfangsposition zurück
